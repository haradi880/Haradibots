This chatbot is built using a Sequence-to-Sequence (Seq2Seq) model with LSTM layers to learn conversational patterns from a small, custom QnA dataset.

Open in Google Colab

🔹 How It Works:

Encoder: Converts the user’s input question into a context vector.
Decoder: Takes the context vector and generates an answer word by word.

Training: Model is trained on input/output QnA pairs using sparse_categorical_crossentropy.

Inference: Encoder understands the question, decoder generates new responses.

🔹 Key Features:
✅ Trained on custom QnA dataset in desi style.
✅ Supports word-by-word generation.
✅ Lightweight and CPU-friendly.

Snake Game with Deep Q-Learning (DQN)

This project uses Deep Q-Learning (DQN) to train an AI agent to play the classic Snake game. The agent learns by interacting with the environment and improving its performance over time.

Need Traning Codes:

Open in Google Colab

🔹 How It Works:

DQN Agent: Uses a neural network to approximate Q-values.
Replay Memory: Stores past experiences for stable learning.
Epsilon-Greedy: Balances exploration and exploitation.
...

Download Model Download Code

Training: The agent is trained over multiple episodes using the DQN algorithm with experience replay.

Inference: The trained model plays the game by selecting actions based on learned Q-values.

🔹 Key Features:
✅ Trained using Deep Q-Learning with epsilon decay.
✅ Includes visual snake with gradient color and border.
✅ Supports custom training and real-time game play.
✅ Optimized for CPU and GPU.

📂 Dataset: 3.6 Million Lines of Raw Text

This dataset contains 1.4 GB of raw, unstructured text data with 36 lakh lines (3.6 million rows). It has been specifically curated for training deep learning models, especially conversational chatbots capable of handling natural, human-like conversations.
